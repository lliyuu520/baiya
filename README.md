## 启动说明

### 后端启动

1. 进入后端目录：
   ```bash
   cd by-admin
   ```
2. 启动服务：
   ```bash
   mvn spring-boot:run
   ```

### 前端启动

1. 进入前端目录：
   ```bash
   cd by-ui
   ```
2. 安装依赖：
   ```bash
   yarn install
   ```
3. 启动开发服务器：
   ```bash
   yarn dev
   ```
